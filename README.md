# Python_Automation
# Python Automation

This repository contains simple Python automation projects that I built to practice Python, APIs, email automation, and GitHub Actions.

## Projects

### Weather Information
- Fetches the current weather for a location.
- Displays details like temperature and weather condition.

### Weather Alerts
- Checks weather conditions.
- Sends an alert email when specific weather conditions are met.

### News Headlines
- Fetches the latest news headlines using a News API.
- Sends the news summary through email.

## Technologies Used

- Python
- Requests
- GitHub Actions
- Weather API
- News API

## Files

- `bot.py` - Main Python script
- `weather_alerts.py` - Weather alert program
- `news_scraper.py` - News fetching program
- `requirements.txt` - Required Python packages

## How to Run

1. Clone the repository.
2. Install the required packages.
3. Run any Python file.
